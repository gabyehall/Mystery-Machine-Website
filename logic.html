Our Logic and Coding Choices

Breakdown of Our Logic and Coding

If there is no space on the board where the robot is certain that there is a wumpus, it will:

  • Move up space if there is no possibility of a wumpus, possibility of a hole, definite hole, or the edge of the board there. Otherwise, it will:
    1. Move down to the bottom of the board again if there is no possibility of a wumpus, possibility of a hole, or definite hole in the way. Otherwise, it will:
    2. Move right space if there is no possibility of a wumpus, possibility of a hole, definite hole, or the edge of the board there. Otherwise, it will:
    3. Move left a space.
  • Take in a sense from the user. If there is nothing there, space and surrounding spaces will be marked as safe. Else:
    1. If there is a breeze sensed, the surrounding boxes will be marked as possible holes, as long as they were not previously marked as safe.
    2. If there is a smell sensed, the surrounding boxes will be marked as possible wumpus, as long as they were not previously marked as safe.
    3. If there is a glitter sensed, the surrounding boxes will be marked as possible gold, as long as they were not previously marked as safe.
    4. If there is no breeze sensed, the surrounding boxes will be marked as not-possible holes.
    5. If there is no smell sensed, the surrounding boxes will be marked as not-possible wumpus.
    6. If there is no glitter sensed, the surrounding boxes will be marked as not-possible gold.
  • The robot will scan through the entire board and adjust spaces based on:
    1. If space is marked as ‘not possible’ for a hole, wumpus, or gold, then and corresponding possibles will be removed from that space.
    2. Any spaces previously marked as ‘safe’ will remove any other marks.
    3. Any spaces that have been marked as possible wumpus twice will be changed to a definite wumpus.
    4. If there are two or more possibles of a certain type on one space, and that space is on an edge border, then the possibility is changed to certain.
    5. If a definite hole was determined to be in a specific space, any other possibilities that came from that hole are removed from the board.
    6. If a definite wumpus was determined, all possible wumpus is removed from the board.
    7. If a possible hole, wumpus, or gold was on space and the other possibility was removed, then that possibility becomes definite.
  • If the robot is on the same tile as the definite gold, then the robot will stop.

    • If there is a space on the board where the robot is certain that there is a wumpus, it will:

  • Check to see if it is currently in a space surrounding the wumpus. If it isn’t:
    1. It will move closer to the wumpus’ current location, prioritizing getting to the same row and column of the wumpus as long as there is no possibility of a wumpus, possibility of a hole, definite hole, or the edge of the board there.
  • If it is:
    1. The robot will face the tile that the wumpus is on and shoot at it.
