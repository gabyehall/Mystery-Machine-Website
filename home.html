About Our Project

Mystery Inc.

Why not kill the Wumpus?

The Wumpus World challenge is an academic endeavor that combines logic, engineering, and collaboration to solve a problem. Our team went about building and coding a robot to successfully navigate a grid full of potential hazards and achieve a set goal. In the Wumpus world challenge, there is a grid with a set amount of holes, 1 Wumpus monster, and 1 pile of gold. If our robot moved to an area containing a hole or the wumpus, we lost the challenge, but if we successfully navigated the grid and found the pile of gold, we won. We can not see where these things are, so we must rely on input from the environment. When a hole is near, a breeze is felt, when the Wumpus is near, there is a smell, and when the gold is near, we see the glitter. When we moved to a new square, we would be informed of any new input, which we would then send to the robot.
